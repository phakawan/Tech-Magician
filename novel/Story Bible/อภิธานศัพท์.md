# อภิธานศัพท์ (Canon)

> เป้าหมาย: ศัพท์เฉพาะสะกดให้คงที่ และความหมายไม่เปลี่ยนกลางเรื่อง

| คำ | ความหมาย | ใช้เมื่อ | ห้ามใช้แทนคำว่า |
|---|---|---|---|
| อาเคนเนีย | นครรัฐเวทมนตร์ศูนย์กลางมาตรฐาน | กล่าวถึงเมืองหลวง/ระบบรัฐ | (สะกดอื่น) |
| โคโรบา | เมืองเล็กทางใต้ จุดเริ่มของอีวาน | ฉากบ้านเกิด/งานช่าง | (สะกดอื่น) |
| เมอริดัส | เมืองท่า/ฐานนักล่าเงินรางวัล | ปมใต้ดิน/อาวุธเถื่อน/เวจ | (สะกดอื่น) |
| จอมเวทสูงสุด | ศูนย์รวมอำนาจและศรัทธาแห่งอาเคนเนีย | การเมือง/พิธีการ/คำสั่งรัฐ | ผู้นำสภา (ถ้าไม่ใช่) |
| ผู้อาวุโสทั้งห้า | แกนอำนาจยาวนานในอาเคนเนีย | สภา/การตัดสินใจเชิงยุทธศาสตร์ | - |
| มิราเคิลเมจ | หน่วยหัวกะทิกำลังรบพิเศษ (อันดับ 1–7) | กล่าวถึงลำดับชั้นกำลังรบ | จอมเวทธรรมดา |
| ไม้กายสิทธิ์ | อาวุธคู่กายจอมเวท (ผูกเลือด+กระแสจิต) | ระบบเวทและการต่อสู้ | คทา |
| คทา | เครื่องมือเวทเฉพาะ (เช่นสายแพทย์ของเฮอร์มาน) | เมื่ออยากแยกจากไม้กายสิทธิ์ | ไม้กายสิทธิ์ |
| กระแสจิต | เงื่อนไขการใช้งานไม้กายสิทธิ์ | อธิบายการ “เข้ากันได้” กับไม้ | พลังเวท (ถ้าหมายถึงคนละอย่าง) |
| เจตจำนงของไม้ | จุดประสงค์เชิงการออกแบบของไม้แต่ละอัน | เวลาอธิบายแนวคิดอีวาน | “วิญญาณไม้” (ถ้าไม่อยากลี้ลับ) |
| เส้นประสาทของไม้ | ส่วนที่ 4 ที่อีวานเพิ่ม | เวลาอธิบายนวัตกรรม/ประสิทธิภาพ | แก่น/เปลือก/อัญมณี |
| ไม้เฉพาะทาง | ไม้ใช้แล้วทิ้งที่บันทึกเวทไว้ | ตัวเปลี่ยนเกม (ช่วยชีวิต/อาวุธ) | ไม้ทั่วไป |
| เอชเชล่อน | องค์กรใต้ดินผลิต/ค้าอาวุธเถื่อน | ปมมืดช่วงบท 1–17 | - |
| สำนักปักษามนตรา | สำนักนอกรีตสำคัญ (เควทซัลสังกัด) | ปมอุดมการณ์/ประวัติศาสตร์ | “กลุ่มผู้ร้าย” (ถ้าอยากให้มีมิติ) |
| จักรวรรดิมืด | มหาอำนาจทางเหนือ ฝ่ายศัตรูรัฐ | สงคราม/การเมืองระหว่างรัฐ | - |
